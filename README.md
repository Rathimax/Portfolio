## ✨ Key Highlight: Interactive Particle Typography
The portfolio features a custom-built, high-performance canvas typography engine. The Name in the Hero section is rendered as thousands of interactive particles that:
- **React to Cursor**: Particles are repelled by mouse movement in real-time.
- **Ambient Life**: Features 60+ floating "ambient" particles that drift naturally, creating a living digital environment.
- **Responsive Auto-Wrap**: Dynamically wraps text into multiple lines on mobile to preserve legibility and impact.
- **Zero-Clipping Buffer**: Uses an oversized canvas buffer to allow particles to fly beyond original bounds without being cut off.

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Rathimax/Portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
